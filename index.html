HTML 기본 문법

html에 대한 설명이 매우 많습니다 여기에 다 쓰어져있는거에영
li 태그 : 점을 찍어줌. 리스트처럼 보이게함. List의 약자
ul 태그 : li를 그룹화지어주는 태그. Unordered List의 약자
ol 태그 : li태그, 목차들에 숫자를 자동적으로 넣어줌 Ordered List 의 약자
title 태그 : 검색엔진과 크롬 위에 탭에 제목을 넣어줌. 마치 표지처럼
meta 태그 : 웹 홈페이지와 속성을 맞춰준다? 만약 서로 다르면 한글이 안나옴. 근데 이거 atom은 되나봄 ㅇㅅㅇ. 본문이 utf-8방식으로 저장되어있따!

매우 중요합니다!
body태그와 head태그 : 프로그래머들 사이의 약속이며 제목과 본문을 구별지어주는 매우 중요한 것임. 꼭 써야함. 약속이기 때문에.
html태그 : body 태그와 head 태그의 어머니로 둘을 잡아주는 태그임. 이것도 써야함.
!doctype html 태그 : 이 문서는 html로 작성되었음을 알려주는 태그 써야함.


이 모든것은 여기에서 결정됩니다 무려 세계국제기구인 W3C라는 곳에서 말이죠!
a태그 : a 태그는 링크 태그이며 자세한 것은 위의 링크에서 설명되어있습니다.
속성중 target은 blank와 parent 등이 있는데 blank가 새탭으로 열기임.
title은 하이퍼텍스트위에 마우스를 올려두면 툴팀(설명)이 뜨는것임.
